# 🐦 X Follow Card - React Component

A faithful, responsive, and highly reusable recreation of the **X (Twitter)** "Who to follow" component. This project showcases fundamental **React** concepts and modern UI development patterns.

## 🚀 Key Features
*   **Interactive State:** Follow button with dynamic style changes (Follow/Following/Unfollow).
*   **Component Composition:** Clean architecture based on the reusable `XFollowCard` component.
*   **Pixel-Perfect Design:** CSS styles that mimic the typography, spacing, and hover effects of the original platform.
*   **Dynamic Avatars:** Profile image generation using the `userName` prop.

## 🛠️ Tech Stack
*   **React.js** (Vite for fast tooling)
*   **JavaScript (ES6+)**
*   **CSS3** (Flexbox and Custom Properties)

## 🧠 Technical Concepts Implemented

This project served as a deep dive into React's core pillars:

1.  **Hooks (`useState`):** Managing local state to toggle follow status, ensuring a reactive and snappy UI.
2.  **Props & Children:** Using props to customize individual cards and `children` to inject flexible content.
3.  **List Rendering:** Utilizing `.map()` to iterate over data arrays and render multiple components efficiently.
4.  **Event Handling:** Managing user interactions to update the Virtual DOM optimally.

## 📦 Installation & Setup

1. Clone the repository:

2. Navigate to the directory:
   cd x-follow-card

3. Install dependencies:
   npm install

4. Start the development server:
   npm run dev
